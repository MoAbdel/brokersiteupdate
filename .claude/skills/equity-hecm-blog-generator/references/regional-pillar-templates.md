# Regional Pillar Page Templates (5,500-6,500 Words)

## Purpose

Regional pillar pages are the **authority hub** for each geographic region per content track. They sit between the state-level pillar and the individual hub posts in the content hierarchy.

```
State Pillar (e.g., "Reverse Mortgage California Guide")
  └─ Regional Pillar (e.g., "Reverse Mortgage Bay Area Peninsula Guide") <-- THIS
       └─ Hub Posts (e.g., "Reverse Mortgage Atherton, Hillsborough & More")
```

## Content Composition: 50% Structured / 50% Prose

Regional pillars are Bing-dominant pages that establish topical authority through comprehensive data and strategic linking.

## Schema Stack for Regional Pillars

All regional pillar pages require these 7 schema types:

### 1. Article Schema
- **HECM track:** Use articleHECM type with reverse mortgage specific properties
- **Equity track:** Use articleEquity type with home equity specific properties
- **Wholesale track:** Use generic article schema
- Include Bing-specific properties: `datePublished`, `dateModified`, `author`, `publisher`
- `speakable` property pointing to citation hook and expert summary selectors

### 2. FAQPage Schema
- 12-15 questions minimum for regional pillars
- Each question must include the region name for local relevance
- Answers formatted as concise, speakable text (under 75 words each)

### 3. LocalBusiness Schema
- `areaServed`: Array of all cities in the region
- `serviceArea`: GeoCircle or GeoShape covering the region
- Include `hasOfferCatalog` with relevant loan products

### 4. Speakable Schema
- Point to: citation hook, expert summary, and top 3 FAQ answers
- Use CSS selectors: `.citation-hook`, `.expert-summary`, FAQ section headings

### 5. BreadcrumbList Schema
- 4-level breadcrumb structure:
  ```
  Home > Blog > [State] Guide > [Region] Guide
  ```
- Example:
  ```
  Home > Blog > Reverse Mortgage California > Reverse Mortgage Bay Area Peninsula
  ```

### 6. ItemList Schema
- Encode the city comparison mega-table as structured data
- Each city as a `ListItem` with `position`, `name`, `description`
- Include property values as `additionalProperty`

### 7. HowTo Schema
- Encode the qualification steps or application process
- HECM: "How to Get a Reverse Mortgage in [Region]"
- Equity: "How to Access Home Equity in [Region]"
- Wholesale: "How to Work with a Wholesale Mortgage Broker in [Region]"

## Internal Linking Requirements for Regional Pillars

### Links OUT (minimum 12):

**To Hub Posts (3-6 links):**
- Link to every hub post that falls within this region

- Place in the opening paragraphs or Regional Market Overview

- Same region, different content track
- Example: HECM regional pillar links to Equity regional pillar for same region
- Anchor text: "Also explore [Region] home equity options" or "Learn about wholesale mortgage rates in [Region]"

**To Adjacent Regional Pillars (2-3 links):**
- Same track, neighboring geographic region
- Example: Bay Area Peninsula HECM pillar links to South Bay HECM pillar
- Anchor text: "Compare reverse mortgage options in neighboring [Adjacent Region]"

**To Existing OC Content (2-3 links):**
- Bridge new geo-expansion content back to established Orange County pages

- State pillar page must link down to this regional pillar
