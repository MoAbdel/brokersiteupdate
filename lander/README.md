# Home Equity Landing Page

Drop this folder into a Next.js App Router project.

## Files

- `app/home-equity/page.tsx`
- `app/home-equity/page.module.css`
- `app/home-equity/opengraph-image.tsx`
- `app/home-equity/twitter-image.tsx`
- `.env.example`

## Install

1. Copy `app/home-equity` into your existing app directory.
2. Copy `.env.example` to `.env.local`.
3. Replace the Figure and Aven affiliate links.
4. Confirm licensing/contact values.
5. Run `npm run build`.

The route will be available at `/home-equity`.
